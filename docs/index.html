Presentación y Licencia

0.1 Presentación

Este curso tiene como principal objetivo enseñar el lenguaje de programación R desde cero para realizar análisis de datos cuantitativos en las ciencias psicológicas. La idea es que el texto sea ágil y fácil de leer, revise conceptos básicos de programación al igual que código esencial para interactuar con R. Se utilizan ejemplos y casos de estudios reales en psicología para ilustrar su uso y cómo aplicarlos en contextos prácticos. Aunque el texto está dirigido a estudiantes de pregrado, de postgrado y/o profesionales de la psicología, o cualquier persona más allá de las fronteras de esta disciplina se puede beneficiar de este curso ya que los principios del uso de R se mantienen constantes independiente del quehacer profesional particular de cada uno. No se necesitan conocimientos en programación y sería deseable que el lector tenga conocimientos básicos de un curso de estadística.
